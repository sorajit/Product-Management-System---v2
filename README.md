### ระบบจัดการสินค้าอย่างง่าย ที่เชื่อมต่อกับฐานข้อมูล
พัฒนาเพิ่มเติมจาก https://github.com/sorajit/Product-Management-System ไปเชื่อมต่อฐานข้อมูลที่สร้างโดยใช้ postgreSQL 
ในการทดสอบในตารางชื่อ "product" ซึ่งมีประกอบด้วย column ={product_id, product_name, category, price, stock}  
โดยมี product_id เป็น primary key
